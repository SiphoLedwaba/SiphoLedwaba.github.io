Project 1: Jobs API

Leveraging the Stack Overflow Developer Survey 2019 subset, this project focused on rigorous data wrangling to ensure dataset integrity. The Current Technology Usage Dashboard includes:

  • Top 10 Languages (Bar Chart): Highlights prevalent programming languages among developers.
  • Top 10 Databases (Column Chart): Visualizes popular databases used by developers.
  • Platforms and WebFrames (Word Cloud and Hierarchy Bubble Chart): Offers insights into platform diversity and web framework preferences.
The Future Technology Trends Dashboard anticipates language and database trends for the upcoming year, and the Demographics Dashboard provides an overview of gender diversity, global reach, and age demographics among respondents.

Project 2: US H1B1 Visa Applications

Working with three H1B datasets (2021, 2022, 2023), this project analyzed approval rates, identified top employers, and calculated rates by city/state using SQL queries. The goal was to provide a comprehensive understanding of H1B visa trends, assisting in decision-making processes related to employment and immigration.

Project 3: Superstore Sales

Carefully analyzing Superstore sales data, this project extracted actionable insights for improved sales strategies and operations. The detailed examination included evaluating current performance, identifying growth opportunities, analyzing customer behavior, and defining key performance indicators for strategic improvements.

Project 4: Waze

Using machine learning models, this project aimed to predict user retention in Waze. The approach involved processing data, creating features, and utilizing Random Forest and XGBoost models with hyperparameter tuning. The impact was enhancing Waze's understanding of user behavior for proactive retention strategies, overcoming challenges such as handling missing values and addressing imbalanced classes.

Project 5: Gamestop & Tesla

This project created Dash dashboards to visualize Tesla and GameStop stock prices and revenue data. The approach included data extraction using yfinance and web scraping from Macrotrends. Separate Dash apps were developed for each stock with interactive charts, enabling visual analysis and comparison of stock prices and revenue trends. The project also addressed challenges in web scraping errors and outlined future steps for improvement.

Project 6: HR Analytics

The HR Analytics Dashboard project provides insights into employee demographics, satisfaction levels, and factors influencing attrition. It presents key metrics, summary information, and actionable insights, empowering HR professionals to make informed decisions for workforce management and retention strategies. The interactive and visually appealing nature of the dashboard facilitates data exploration and enhances understanding of the HR analytics dataset.

Project 7: Cryptocurrency

This cryptocurrency trading data analysis project aims to generate compliance metrics and interactive reports. Data is loaded, cleaned, and processed, with reports on Daily Trading Volume and Volatility saved as CSV files. Interactive reports for specific tickers are created using seaborn and Plotly, along with a dropdown widget for dynamic updates.
